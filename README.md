Freelancing Platform
A full-stack freelancing platform that connects Clients and Freelancers through project bidding and project management workflows. The application includes role-based access for Admin, Client, and Freelancer users.

🚀 Features
Client
Register and Login
Create Projects
View Own Projects
View Freelancer Bids
Accept or Reject Bids
Track Project Status
Freelancer
Register and Login
Browse Available Projects
Place Bids on Projects
View My Bids
Complete Assigned Projects
Admin
Manage Users
Manage Projects
Manage Bids
Monitor Platform Activities

⚙️ Installation & Setup
Backend
Open Backend project in Eclipse/STS
Configure MySQL Database
Update application.properties
Run Spring Boot Application
Frontend
Open Frontend folder in VS Code

Install dependencies

npm install
Start application
npm start
Frontend URL:

http://localhost:3000
Backend URL:

http://localhost:8080
